<h2 id="overview">Overview</h2>

This guide outlines the step-by-step procedure to extract and export layers from an AutoCAD project and process them using GIS and web applications to complete the BOM (Bill of Materials). The workflow involves using AutoCAD, QGIS, and the Djangomatic platform.

---

<h2 id="requirements">Requirements</h2>

Before starting, ensure you have the following:

- **AutoCAD** software with MAPEXPORT functionality.
- **Export Profiles** for Points ('xplore_points_exp_profile.epf') and Lines ('xplore_lines_exp_profile.epf').
- **QGIS** (latest stable version recommended).
- The Python script 'BOM_Data_Processing.py'.
- A folder to store the extracted shapefiles (e.g., 'BOM_EXTRACT').
- Access to the **Djangomatic** platform with an SSO account.
- The BOM template for pasting the summary data.

---

<h2 id="step-1-open-autocad-project">Step 1: Open the AutoCAD Project</h2>

1. Launch **AutoCAD**.
2. Open the project from which you need to extract layers.

---

<h2 id="step-2-extract-layers">Step 2: Extract Layers</h2>

1. Use the command **MAPEXPORT**.
2. Perform two separate exports:
   - For the **POINTS** features, load the profile 'xplore_points_exp_profile.epf'.
   - For the **LINES** features, load the profile 'xplore_lines_exp_profile.epf'.
3. Export the respective features using the loaded profiles. These profiles are pre-configured to extract the required layers and attributes for Points and Lines.

---

<h2 id="step-3-open-qgis">Step 3: Open QGIS</h2>

1. Launch **QGIS**.

---

<h2 id="step-4-open-python-console">Step 4: Open the Python Console</h2>

1. Access the Python Console:
   - In the top menu, go to **Plugins > Python Console**.

<LoadDynamicImage imageName="qgis_python_console" width={700} />

*Accessing the Python Console in QGIS.*

---

<h2 id="step-5-open-script-editor">Step 5: Open the Script Editor</h2>

1. Open the **Script Editor**:
   - In the Python Console, click the **Show Editor** button (paper and pencil icon).

<LoadDynamicImage imageName="qgis_script_editor" width={700} />

*Opening the Script Editor in the QGIS Python Console.*

---

<h2 id="step-6-load-script">Step 6: Load the Script</h2>

1. Load the script into the editor:
   - Click the **Open Script** button (folder icon).
   - Navigate to the location of the 'BOM_Data_Processing.py' script.
   - Select the file and click **Open**.

<LoadDynamicImage imageName="qgis_load_script" width={700} />

*Loading a script into the QGIS Script Editor.*

---

<h2 id="step-7-run-script">Step 7: Run the Script</h2>

1. Run the script:
   - Click the **Run Script** button (green play icon).
2. The script will prompt for the **POINTS** and **LINES** shapefiles and process them into separate shapefiles needed for the BOM app in Djangomatic.

<LoadDynamicImage imageName="qgis_run_script" width={700} />

*Running a script to process extracted layers in QGIS.*

---

<h2 id="step-8-compress-shapefiles">Step 8: Compress Shapefiles into a ZIP File</h2>

1. Navigate to the folder containing the extracted shapefiles (e.g., 'BOM_EXTRACT').
2. Select all the shapefiles.
3. Right-click and choose **Send to > Compressed (zipped) folder**.
4. Name the ZIP file appropriately.

<LoadDynamicImage imageName="compress_shapefiles" width={700} />

*Compressing shapefiles into a ZIP file for upload.*

---

<h2 id="step-9-upload-zip-djangomatic">Step 9: Upload and Process the ZIP File on Djangomatic</h2>

1. Go to the **Djangomatic** website.
2. Log in using SSO.
3. Under the **Apps** section, open the **XPLORE** drop-down menu and select the **Extract BOM Summary** app.
4. Upload the ZIP file containing the shapefiles.
5. Click **Start Task** and wait for the process to complete.
6. Download the generated ZIP file containing the summary.

<LoadDynamicImage imageName="djangomatic_upload" width={700} />

*Uploading and processing a ZIP file on Djangomatic.*

---

<h2 id="step-10-copy-summary-bom-template">Step 10: Copy and Paste Summary into BOM Template</h2>

1. Open the BOM template.
2. Copy the relevant data from the extracted Excel summary.
3. Paste the data into the BOM template under the appropriate FSA sheet and segment.

<LoadDynamicImage imageName="bom_template_copy" width={700} />

*Copying data from the summary into the BOM template.*
